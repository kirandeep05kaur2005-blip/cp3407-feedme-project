# FeedMe - Food Ordering System

## Overview
FeedMe is a simple web-based food ordering system developed using PHP and MySQL. Users can browse restaurants, view menus, add items to a cart, and place orders.

## Features
- Browse restaurants
- View menus
- Add/remove items from cart
- Place orders
- View order history

## Technologies Used
- PHP
- MySQL
- Bootstrap 5
- XAMPP

## How to Run
1. Install XAMPP
2. Place project in:
   C:\xampp\htdocs\feedme
3. Start Apache and MySQL
4. Open:
   http://localhost/feedme/
