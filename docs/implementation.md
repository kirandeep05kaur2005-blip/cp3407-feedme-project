## Iteration 2 Update
The cart system was improved to support saving orders into the database. When the user clicks the Place Order button, a new order is stored in the `orders` table and each selected cart item is stored in the `order_items` table. This improved the project by making the ordering process functional instead of static.
The system allows users to browse restaurants, view menus, add items to a cart, and place orders. When an order is placed, data is stored in the `orders` and `order_items` tables, demonstrating full backend integration.
