# Requirements

## Project Title
FeedMe: A Web-Based Food Ordering System

## Problem
Many food ordering apps are confusing or too complex. This project creates a simple and easy-to-use food ordering website.

## Project Goal
The goal is to build a website where users can browse restaurants, view menus, add items to a cart, and place orders.

## Users
- Customer
- Admin

## Features

### Customer Features
- View restaurants
- View menu items
- Add items to cart
- Update cart
- Place order
- Register
- Login
- View order history

### Admin Features
- Add restaurants
- Edit restaurants
- Delete restaurants
- Add menu items
- Edit menu items
- Delete menu items
- View orders

## User Stories
1. As a customer, I want to browse restaurants so I can choose food.
2. As a customer, I want to view menu items so I can decide what to order.
3. As a customer, I want to add food to cart so I can prepare my order.
4. As a customer, I want to place an order so I can buy food online.
5. As a customer, I want to register and login so my data is saved.
6. As a customer, I want to view order history so I can see past orders.
7. As an admin, I want to manage restaurants so data stays updated.
8. As an admin, I want to manage menu items so menus are correct.
9. As an admin, I want to view orders so I can manage the system.
